DEMOLISHER

Edit, Play and Render .demo files recorded in Cryptic Studios games

Games you can make .demo files with:

Champions Online http://co.perfectworld.com/

Neverwinter http://nw.perfectworld.com/

Star Trek Online http://sto.perfectworld.com/

To Record a .demo file:

Play one of the games.

Type /demo_record yourDemoName into the game's chat window

recording demo should appear on screen

To stop recording:

Type /demo_record_stop into the game's chat window and press return.

A text file will be saved to your game folder's /Live/demos/ directory with the name you gave it and the .demo extension.

Find .demo files on your PC:

Type "Live/demos" into the Search field of the Windows Start menu.

The path is typically something like:

"C:\Program Files (x86)\Cryptic Studios\Champions Online\Live\demos"

Tell Demolisher where your .demo files are:

Drag a .demo file to the Demolisher.exe icon or onto it's open window.

Tell Windows to open .demo with Demolisher by default:

Right click a .demo file in Windows Explorer and choose Properties from the menu.

Choose Change from the "Opens With" option and browse to Demolisher.exe

If this Demolisher.exe does not work with your file, try the default method.

Default Playback:

Run the game's launcher and choose Options

Enter -demoplay yourDemoName.demo into the Advanced Command Line:

Login and hit play button to watch playback and/or edit your demo.
